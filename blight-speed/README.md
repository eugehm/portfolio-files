# Glove Assets

Asset sample from [Blight Speed](https://kilarivi.itch.io/blight-speed), a game made for Michigame Jam.

**glove-material.png** is the texture applied to both gloves.

**left-glove.blend/left-glove.fbx** and **right-glove.blend/right-glove.fbx** are the left and right glove 3D models with animations. The .blend files are in Blender's native format, while the .fbx files are exported for use in the game engine.
